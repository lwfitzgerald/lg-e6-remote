# lg-e6-remote
Interactive remote control for LG E6 TV

## Running it
```
export TV_IP_ADDRESS=192.168.0.2

npm install
node e6-remote.js $TV_IP_ADDRESS
```

## Supported controls
* ```s```: Open white balance menu
* ```m```: Open app launcher
* ```Arrow keys```
* ```Enter```
* ```Esc/Backspace```: Back
